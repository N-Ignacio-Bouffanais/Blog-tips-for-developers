Welcome future developer

In this page you will find a lot of tips for become a software developer/Frontend developer. I will try to explain how to start with different approaches, which videos to watch, which tutorials have helped me, code repositories, which technologies to learn and how not to get lost along the way.

JavaScript

-JavaScript es uno de los lenguajes de programacion mas utilizados a nivel mundial, principalmente se usa en el lado del cliente, y gracias a el podemos crear sitios web dinamicos. Aprender JS es crucial para convertirse en desarrollador Web o Frontend developer.

JS como lenguaje universal

Con el paso del tiempo JavaScript se a comenzado a utilizar como lenguaje multiproposito, en otras palabras, este se puede utilizar en otras areas del desarrollo de software;

  1. -Ingenieria: Se pueden programar chips y pequeñas placas electronicas como arduino y Raspberry Pi, entre otras.
  2. -Aplicaciones moviles: Con JavaScript podemos crear aplicaciones que interactuen directamente con el sistema operativo de los celulares, esto especifcamente se hace frameworks o librerias de JavaScript como Reactjs
  3. -Servidores: Con Nodejs podemos crear servidores de alta fiabilidad y seguridad que reciban y procecen multiples peticiones de una aplicacion.
  4. -Amoung other topic such as Machine learning and Web 3.

CSS

Frameworks

CSS has several frameworks, which are a way of styling websites in a simpler and faster way. The 2 most popular are:

  1. TailwindCSS

    It stands out for its simplicity of learning and use, it is a great option if you want to have control over most of the styles of your websites.

  2. Bootstrap

    It focuses on giving you out-of-the-box components such as navigation bars, menus, buttons, tables, cards, and more things... The big drawback is the lack of control over styles, in other words, web pages designed with Bootstrap look a lot like each other.
